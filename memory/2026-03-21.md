- **Bore tunnel**: bore.pub:11710 → localhost:9091 (ephemeral, changes on restart)
- **Bore binary**: `/tmp/bore-bin/bore` (v0.6.0)
- **Model download trick**: Must use `-H "User-Agent: Mozilla/5.0..."` AND `-H "Referer: https://katagotraining.org/networks/"` to avoid 403

### Restart procedure (if pod restarts):
```bash
# 1. Restart Go server
nohup python3 /tmp/go_server2.py > /tmp/go2.log 2>&1 &
# 2. Restart bore
nohup /tmp/bore-bin/bore local 9091 --to bore.pub > /tmp/bore2.log 2>&1 &
# 3. Get new URL from: cat /tmp/bore2.log
```
Note: All /tmp files are lost on pod restart — need to re-download katago binary, model, and bore.
